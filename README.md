Program yang saya buat di gunakan untuk menghitung faktorial dari angka yang di input oleh user. 
Cara kerja program ini adalah user menginput angka lalu komputer akan menjalankan program yaitu menghitung dengan mengalikan angka dari angka 1 sampai bilangan yang di input ole user.
Selanjutnya program akan mengeluarkan output berupa Hasil dari faktorial angka tersebut.
